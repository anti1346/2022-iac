## 플레이북 기본 문법 익힘
- https://docs.ansible.com/ansible/latest/reference_appendices/playbooks_keywords.html

#### play
```
ansible mgmt -m copy -a 'src=hello.txt dest=/tmp/hello.txt backup=true'
```
ad-hoc 명령을 playbook으로 변환
```
- hosts: mgmt
  tasks:
    - copy: 
      src: hello.txt
      dest: /tmp/hello.txt
...
```
